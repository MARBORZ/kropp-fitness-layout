# Kropp Fitness — Layout Eksperiment

En layout-fokusert fitness landingsside bygget med semantisk HTML, moderne CSS og vanilla JavaScript. Vektlegger visuelt hierarki, typografi og responsivt design uten rammeverk.

## 🎨 Om

Kropp Fitness er et omfattende landingsside-eksperiment som viser frem moderne CSS-layoutteknikker, semantisk HTML-struktur og rene designmønstre. Har flere seksjoner inkludert hero-banner, motivasjonskort, BMI-kalkulator, videointegrasjon og responsiv navigasjon.

## ✨ Funksjoner

- **Semantisk HTML5** — Riktig dokumentstruktur og tilgjengelighet
- **Moderne CSS Layouts** — Flexbox og Grid for responsivt design
- **Tilpasset Typografi** — Heebo og Yantramanav skriftfamilier
- **Interaktive Skjemaer** — BMI-kalkulator og abonnementsskjemaer
- **Videointegrasjon** — Innebygd video med tilpassede kontroller
- **Responsivt Design** — Mobil-først tilnærming med adaptive layouts
- **Visuelt Hierarki** — Backdrop-titler og strukturerte innholdsseksjoner
- **Sosiale Medier Lenker** — Integrerte sosiale plattform-koblinger

## 🛠️ Teknologi

- **HTML5** — Semantisk markup og struktur
- **CSS3** — Moderne layoutteknikker (Flexbox, Grid)
- **Vanilla JavaScript** — Ingen rammeverk eller biblioteker
- **Google Fonts** — Heebo og Yantramanav skrifttyper

## 🚀 Kom i Gang

Åpne ganske enkelt `index.html` i en nettleser:

```bash
# Bruk en lokal server (anbefalt)
npx serve .

# Eller åpne direkte
open index.html
```

## 📁 Struktur

```
kropp-fitness-layout/
├── index.html          # Hoved HTML-fil
├── css/
│   ├── normilize.css   # CSS reset
│   └── style.css       # Hovedstiler
├── js/
```
