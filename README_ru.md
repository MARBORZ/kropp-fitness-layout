# Kropp Fitness — Эксперимент с Макетом

Фитнес-лендинг с фокусом на макет, построенный на семантическом HTML, современном CSS и vanilla JavaScript. Акцент на визуальной иерархии, типографике и адаптивном дизайне без фреймворков.

## 🎨 О Проекте

Kropp Fitness — это комплексный эксперимент с лендингом, демонстрирующий современные техники CSS-макетов, семантическую HTML-структуру и чистые паттерны дизайна. Включает множество секций: hero-баннер, мотивационные карточки, BMI-калькулятор, видео-интеграцию и адаптивную навигацию.

## ✨ Возможности

- **Семантический HTML5** — Правильная структура документа и доступность
- **Современные CSS Макеты** — Flexbox и Grid для адаптивного дизайна
- **Кастомная Типографика** — Шрифтовые семейства Heebo и Yantramanav
- **Интерактивные Формы** — BMI-калькулятор и формы подписки
- **Видео Интеграция** — Встроенное видео с кастомными контролами
- **Адаптивный Дизайн** — Подход mobile-first с адаптивными макетами
- **Визуальная Иерархия** — Backdrop-заголовки и структурированные секции контента
- **Ссылки на Соцсети** — Интегрированные подключения к социальным платформам

## 🛠️ Технологии

- **HTML5** — Семантическая разметка и структура
- **CSS3** — Современные техники макетов (Flexbox, Grid)
- **Vanilla JavaScript** — Без фреймворков и библиотек
- **Google Fonts** — Шрифты Heebo и Yantramanav

## 🚀 Начало Работы

Просто откройте `index.html` в браузере:

```bash
# Используя локальный сервер (рекомендуется)
npx serve .

# Или откройте напрямую
open index.html
```

## 🎯 Ключевые Секции

- **Header** — Навигация с логотипом и бургер-меню
- **Banner** — Hero-секция с информацией о событиях
- **Motivation** — Двойной макет карточек с backdrop-заголовками
- **Gym Logos** — Секция демонстрации партнеров
- **Join Us** — Видео-секция с формой подписки
- **Map** — Секция поиска локации
- **Family** — Демонстрация фотогалереи
- **Calculate** — BMI-калькулятор с интерактивной формой
- **Footer** — Многоколоночный макет с контактной информацией и социальными ссылками
